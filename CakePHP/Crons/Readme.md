### Create a Cron on Combell
Open Mremote and check in the root folder if project has any crons. \
To create a cron:
```
cat .crontab
nano .crontab
```
To save:
```
Ctrl - O 
```
To close:
```
Ctrl - x
```
Always test your Cron!!
